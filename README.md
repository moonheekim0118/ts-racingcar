## 타입스크립트 레이싱경주

### 기능 구현

- [] 자동차의 이름을 입력 받는다.

  - [] 자동차의 이름은 쉼표단위로 구분된다.
  - [] 이름은 5자 이하여야 한다.
  - [] 이름은 공백일 수 없다.
  - [] 이름은 중복 될 수 없다.

- [] 레이싱 경주를 시도 할 횟수를 입력받는다.

  - [] 시도 횟수는 양의 정수만 입력 받을 수 있다.

- [] 레이싱 경주 룰을 적용한다.
  - [] 각 자동차에 0 이상 9 이하의 난수를 부여한다.
  - [] 난수가 4이상 일 경우 전진한다.
  - [] 모든 자동차가 전진이 끝나면 경주 결과를 출력한다.
  - [] 위 과정을 시도 횟수만큼 반복한다.
  - [] 경주가 끝나면 우승자를 출력한다.
  - [] 우승자는 중복 될 수 있다.
